TTS Ephemeral направлялся с военной базы Три-Тахион на Куланне, чтобы усилить блокаду системы Вальхалла, когда поступили новости о снятии осады Расвельга. Это было шоком даже для Стратегического Предиктора, который мог предложить только «человеческий фактор» в качестве объяснения недосмотра. Тем не менее, Predictor разработал план действий на случай непредвиденных обстоятельств и разместил его в запечатанных распоряжениях для капитана-директора Ephemeral.

Ephemeral уменьшает мощность двигателя, чтобы избежать обнаружения, и резко меняет курс.

По чистой случайности, патруль дальнего действия Гегемонии обнаружил Эфемераль непосредственно перед тем, как он снова вошел в Атлантическое Молекулярное Облако. Лидер патруля - Байкал Дауд, который поднялся до командного ранга Гегемонии из трущоб Чичомозтока, живуч. Он делает азартную игру и строит курс перехвата, используя последнюю известную траекторию Эфемераля в качестве руководства, по словам отчета о последующих действиях, «где * нет * искать носителя TriTach».

Думая, патруль Дауда будет легкой добычей для его продвинутых кораблей, капитан-директор систем эфемерных приказов, включенных для боевых маневров и развертывания всех истребителей. Отличительный образец высокоэнергетического сенсора могущественного Астрального носителя внезапно появляется из банка молекулярного облака по курсу, направленному прямо к патрулю Гегемонии.

Пришло время мальчику из нижней части Чикомозтока показать этим блестящим корпоративным принцессам, как бороться.
